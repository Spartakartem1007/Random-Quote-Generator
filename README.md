Random Quotes Generator — это программа, которая предлагает пользователю случайную цитату из тщательно подобранной коллекции из 200 мудрых изречений. Каждая цитата подобрана для разных жизненных ситуаций: моментов сомнения, поиска вдохновения, необходимости мотивации или простого размышления о жизни.

Random Quotes Generator is a program that offers users a random quote from a carefully curated collection of 200 wise sayings. Each quote is selected for different life situations: moments of doubt, seeking inspiration, needing motivation, or simply contemplating life.
